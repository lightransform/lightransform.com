
Lightransform - Unicode to Walkman-Chanakya 905 Converter

FEATURES:
- Upload image with handwritten/printed Hindi
- OCR extracts Hindi text using Tesseract.js
- Real Walkman encoding using walkman.tsv mapping
- Outputs directly in one box
- Copy and Clear buttons for convenience

HOW TO DEPLOY (on Vercel):
1. Sign up at https://vercel.com/
2. Create a new project and import this folder
3. Set it to deploy with static hosting (no backend needed)
4. Done! Your site will go live on a custom vercel.app subdomain

Note: You can edit walkman.tsv to fine-tune mappings.
